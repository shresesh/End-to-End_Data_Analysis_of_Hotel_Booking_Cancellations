# Hotel Booking Cancellations: End-to-End Data Analysis

## üè® Project Overview
High cancellation rates significantly impact revenue in the hospitality industry. This project utilizes Python to perform a deep dive into hotel reservation data, identifying the key drivers behind cancellations. By analyzing pricing, seasonality, and customer behavior, this project provides data-driven recommendations to help hotels optimize their revenue and reduce "no-show" rates.

## üìä Key Findings & Insights
* **Price Sensitivity:** Identified a strong positive correlation between higher **Average Daily Rates (ADR)** and increased cancellation rates.
* **Seasonality Trends:** Discovered that cancellations peak during specific months, suggesting a need for dynamic seasonal overbooking strategies.
* **Lead Time Impact:** Analysis revealed that reservations made with longer lead times are significantly more likely to be cancelled.
* **Market Segment Analysis:** Evaluated how different booking channels (Online TA vs. Offline) contribute to the bottom line, highlighting varying risk levels.
